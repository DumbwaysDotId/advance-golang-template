# Prepare

Installation:

- Gorm

  ```bash
  go get -u gorm.io/gorm
  ```

- MySql
  ```bash
  go get -u gorm.io/driver/mysql
  ```

Database:

- Create database named `dumbmerch`

- Create `pkg` folder, inside it create `mysql` folder

- Inside `mysql` folder, create `mysql.go` file
